#   Realtime Chat App

A **full-stack real-time chat application** with secure authentication, real-time messaging, and modern UI.

---

# 🚀 Features

* 🔐 Custom **JWT Authentication**
* ⚡ **Real-time messaging** with Socket.io
* 🟢 **Online / Offline user status**
* 🔔 **Typing & notification sounds**
* 📨 **Welcome emails on signup**
* 🗂️ **Image uploads with Cloudinary**
* 🚦 **API rate limiting**
* 🎨 Modern UI with **React + TailwindCSS + DaisyUI**
* 🧠 State management using **Zustand**
* 🚀 Deployment ready

---

# 🛠 Tech Stack

### Frontend

* React
* Tailwind CSS
* DaisyUI
* Zustand
* Axios
* Socket.io-client

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Socket.io
* Cloudinary
* Resend Email API
* Arcjet Rate Limiting

# 🔧 Run Backend

```
npm install
npm start
```

Server runs on:

```
http://localhost:5000
```

---

# 💻 Run Frontend

```
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
